# 屏幕 / 键盘清洁工具 🧹

一个简洁的纯黑全屏网页工具，用于辅助清洁显示器屏幕和键盘。

> 🌐 **在线使用：** https://kyuuseiryuu.github.io/screen-cleaner/

## 功能特性

| 功能 | 说明 |
|------|------|
| 🖥️ **屏幕清洁模式** | 全屏纯黑显示，方便看清屏幕上的灰尘和污渍 |
| ⌨️ **键盘清洁模式** | 进入后屏蔽所有键盘事件，清洁键盘时不会误触打字 |
| 🔒 **自动全屏** | 进入清洁模式时自动触发浏览器全屏，隐藏地址栏/标签栏 |
| 🌙 **屏幕常亮** | 利用 Screen Wake Lock API 防止清洁时屏幕熄灭 |
| 📱 **移动端支持** | 手机/平板触摸操作同样适用 |

> ⚠️ **注意：** Win 键（Windows 键）是操作系统级别的按键，浏览器无法拦截。如需彻底锁定键盘，建议拔掉键盘再清洁。

## 使用方法

1. 直接打开 https://kyuuseiryuu.github.io/screen-cleaner/ 或本地运行
2. **单击**页面任意位置 → 进入清洁模式（全屏纯黑 + 键盘屏蔽）
3. **双击**页面 → 退出清洁模式
4. 提示文字会在数秒后自动淡出，不干扰清洁

## 使用场景

- **显示器清洁：** 全黑背景让屏幕上的灰尘和指纹一目了然
- **键盘清洁：** 屏蔽按键输入，放心擦拭键盘不会弹出奇怪的东西
- **手机/平板清洁：** 触摸操作同样适用
